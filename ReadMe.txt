JavaBlockBreaker

Summary: Here you have the simple well known blockbreker game with a friendly user interface in the form of a home screen and main game.

In addition to the entry point there are 3 classes:
Gameplay.java
MapGenerator.java
Menu.Java

The gameplay handles user keypressed by responsivly repainting the jpanel.
The MapGenerator simple provides an array with the correct number of rows and collumns for the game to display.
The Menu simple renders a JPanel with a form.
The entry point initalises the main application window using the card layout. It then adds the game and menu to the screen and displays appropreiate one.

For more detailed info look at the comments in the code.
